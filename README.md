# CarPrice

> ML-Powered Car Price Prediction

## Overview

A machine learning model that predicts used car prices based on features like make, model, year, mileage, fuel type, and condition. Compares multiple regression algorithms to find the best predictor.

## Key Features

- **Multiple Models** — Linear Regression, Random Forest, XGBoost, Gradient Boosting
- **Feature Engineering** — Automatic feature extraction and encoding
- **Model Comparison** — Side-by-side accuracy metrics (RMSE, MAE, R²)
- **Web Interface** — Flask app for interactive price estimation
- **Data Visualization** — Feature importance, residual plots, price distributions

## Tech Stack

Python, scikit-learn, XGBoost, Pandas, Flask, Matplotlib

## Getting Started

```bash
git clone https://github.com/MukundaKatta/carprice.git
cd carprice
pip install -r requirements.txt
python main.py
```
